# FP2級 / ライフプランニングと資金計画

## FPの仕事とコンプライアンス

### FPの仕事で守るべきこと

**資金計画のアドバイス** をするのががFPの仕事。

- 顧客の利益を優先
- 守秘義務の遵守
  - 個人を特定できる情報はすべて個人情報保護法の規制対象である
  - 個人情報取扱事業者には、目的の公表または通知の義務がある
- 顧客に対する説明義務 (アカウンタビリティ)

**消費者契約法** は、個人消費者を保護するための法律。

- 追認できる日から**1年間**、または契約から**5年間**が保護対象期間
- **不適切勧誘**で消費者が**誤認・困惑**した場合、契約を取り消せる
- **一方的に不利な条項**は**無効化**できる

**金融サービス提供法** は、個人及び事業者(プロ投資家を除く)を保護するための法律。

- ほぼ全ての金融商品が対象(ゴルフ会員権や金は除く)
- **重要事項の説明をせず、顧客が損失**を被ったならば、販売者は**損害賠償責任**を負う

### FPの仕事の範囲と関連法規

**独占業務**に抵触しないように注意する。
以下はFPが行ってはいけない業務の一覧である。
総じて、個別具体的でない、一般的な説明や助言であればFPでも可能である。

- 税理士法
  - **個別具体的**な税務相談や税務署類の作成
- 弁護士法
  - **個別具体的**な法律相談や法務事務
  - ただし顧客の任意後見受任者になることは可能
- 司法書士法
  - 登記書類の作成や申請代行
- 社会保険労務士法
  - 社会保険に関する書類の作成、提出の代行
- 保険業法
  - 保険の募集や勧誘
- 金融商品取引法
  - 投資判断の助言

### FPの業務アプローチ方法

6つのステップでFPの業務を進める。

- **顧客**との信頼関係の構築
- **顧客**から情報収集
- **顧客**の現状と問題を分析
  - ライフイベント表、キャッシュフロー表、個人バランスシートなどを活用
- **プラン**の提示
- **プラン**の実行援助
- **プラン**の見直し

### ライフイベント表

ライフイベント表は、**将来設計を表にまとめたもの**である。
横軸に経過年数を取り、縦軸に家族、年齢、イベント、必要資金を記入していく。
金額は**現在価値**で入力する点に注意する。

### キャッシュ・フロー表

キャッシュフロー表とは、将来の収支状況と貯蓄残高の推移を表にまとめたものである。
ライフイベント表をもとに作成する。
金額は、変動率などを考慮した**将来価値**で入力する点に注意する。

作成要素は以下の5つ。

- 年間収入
  - 税や社会保険料を引いた後の**可処分所得**で記載する
- 年間支出
- 年間収支
- 貯蓄残高
- 変動率
  - 給与の上昇率や投資信託の利回りなど

### 個人バランスシート

ある時点での資産と負債の状況を見るためのもの。
資産は**時価**で、負債は**残額**で記載する。

生命保険の時価は**返戻金相当額**である。

## 資産計算で使う6つの係数

用途によって使い分ける。いずれも、毎年複利で運用する点は同じ。

- 単純に総額を求めるのが、**終価係数**と**現価係数**。
- 積み立て・取り崩しを前提として、総額を求めるのが、**年金終価係数**と**年金現価係数**。
- **期ごとの金額**を求めるのが、**減債基金係数**と**資本回収係数**。

## 教育資金

**こども保険(学資保険)** は、満期時に満期保険金を受け取れる。
契約者が死亡などした場合に、以後の支払いが免除されるのが特徴。

**教育一般貸付**(通称：国の教育ローン)の特徴は以下の通り。

- 学生1人につき **350万円** (一部450万円)まで借りられる
- 返済期間は**20年以内**
- **固定金利**
- **中学校卒業以上**を対象とする3ヶ月以上の学校
- 子の数に応じた世帯所得の制限あり

**奨学金**には、無利子の**第一種奨学金**と、有利子の**第二種奨学金**がある。
これらは併用できる。
申込みには人的保証と機関保証のいずれかが必要である。

住民税非課税世帯を対象とした**給付型奨学金**もある。
第1種との併用には一定の制限がある。

## 住宅資金

住宅ローンの金利には3種類ある。

- **固定金利型**: 借入期間中ずっと同じ金利
- **変動金利型**: 金利が**半年**ごとに見直される。元利均等の場合、返済額は**5年**ごとに見直される。
- **固定金利期間選択型**: 最初は固定金利で、その後どちらかを選択する。

返済方法は2種類ある。

- **元利均等返済**
  - 毎月の返済額が一定
- **元金均等返済**
  - 毎月の返済額が不定
  - 元本部分は毎月均等で、その上に利子が乗るので、徐々に返済額が減っていく
  - 利子を早く返すため、総返済額はこちらのほうが少ない

借入額を増やす方法として、**ペアローン**と**収入合算**がある。

- ペアローン
  - ローンをそれぞれで契約、手数料も審査も倍
  - 互いに相手の連帯保証人になる
  - どちらも住宅ローン控除や団信を利用可能
- 収入合算
  - ローンは1本
  - 収入合算者のみが連帯保証人になる
  - 住宅ローン控除や団信は、借入者のみが利用可能

住宅ローンの繰り上げ返済には、**期間短縮型**と**返済額軽減型**がある。
前者は利息軽減効果が高い。
後者は毎月の返済額が減るので、家計の負担が減る。

住宅ローンには公的ローンと民間ローンがある。

- 財形住宅融資 (公的ローン)
  - 財形貯蓄を**1年以上**継続し、残高が**50万円以上**あることが条件
  - 貯蓄残高の **10倍 (最高4000万円)** までで、住宅取得額の **90%** が限度
  - 5年固定金利
- フラット35 (民間ローン)
  - **70歳未満**であること
  - 年間返済額が年収の**35%(年収400万円未満は30%)以下**であること
  - 床面積が**70㎡以上、マンションでは30㎡以上**であること
  - 店舗兼用なら**店舗面積が居住面積を超えない**こと
  - 住宅取得額の **100%** までで、**8000万円**が限度
  - 返済期間は**35年**かつ完済時に**80歳以下**であること
  - 固定金利
  - **団信**の費用が含まれている
  - 保証人等は不要
  - 繰り上げ返済は**無料**で、窓口**100万円**、ネット**10万円**から可能
  - **借り換え先としての選択が可能**

**財形貯蓄制度**は、勤労者が給与から天引きで貯蓄する制度。
**一般財形貯蓄**、**財形住宅貯蓄**、**財形年金貯蓄**の3種類がある。
いずれでも、財形住宅融資を利用できる。
申し込み時の年齢は**55歳未満**、積立期間は**5年以上**であることが条件。
財形住宅・財形年金あわせて**550万円**まで利子が非課税である。

**ローンの借換え**を行なうことができるが、団信も含めて再契約になるので、
諸費用を含めて本当に得なのかを見極める必要がある。
なお、**公的ローンへの借り換えはできない**。

## 老後資金

老後資金には、**退職金**、**年金**、**貯蓄**の3つがある。

**リバースモーゲージ**とは、住宅を担保にして融資を受ける制度である。
死亡時には自宅を売却等して一括返済する。
もし債務が残った際に、相続人が返済義務を負う**リコース型**と、負わない**ノンリコース型**がある。

**サービス付き高齢者向け住宅(サ高住)** は、安否確認や生活相談サービスが提供される住宅である。
60歳以上など一定の要件を満たす人が入居できる。

## 社会保険の基本

社会保険は加入が義務付けられた最低限の公的保険である。
**医療保険**、**年金保険**、**介護保険**を指す。
広義だと、**雇用保険**、**労災保険**の2つも含まれる。

- 広義の社会保険
  - 狭義の社会保険
    - 医療保険
      - 健康保険 (協会けんぽ、組合健保)
      - 国民健康保険
      - 後期高齢者医療制度
    - 年金保険
      - 国民年金
      - 厚生年金保険
    - 介護保険
  - 労働保険
    - 労災保険
    - 雇用保険

### 健康保険

健康保険には会社員などが加入する。
協会けんぽ(中小企業)と組合健保(大企業)がある

被保険者と被扶養者が保険の対象になる。
同一生計で、年収が130万円未満、かつ被保険者の年収の1/2未満であれば、被扶養者になれる。
甥なども被扶養者になれるが、その場合は同居が必須となる。

保険料は**労使折半**である。
一般保険料は標準報酬月額(通勤手当も含む)に**一般保険料率**をかけて算出する。
協会けんぽの一般保険料率は都道府県ごとに異なり、組合健保の一般保険料率は組合ごとに異なる。
**介護保険料率**は全国一律である。
育休産休中は、保険料の免除がある。

給付には6つの種類がある。

#### 1. 療養の給付、家族療養費

病院などを**3割負担**で受診できる。
未就学児や、70-74歳で所得が少ない場合は、**2割負担**になる。

#### 2. 高額療養費

同一月の医療費の自己負担額(≒3割の部分)が一定額を超えた場合、**超えた分が支給**される。
差額ベッド代や食事代は対象外である。

限度額は、標準報酬月額28-50万円の人で、以下の通り。

限度額 = **80,100円** + (医療費 - 267,000円) × **1%**

#### 3. 傷病手当金

**被保険者が**病気や怪我で**3日以上連続**して休み、**4日目以降**について給与の支払いがない場合に、
最長**1年6ヶ月を限度**に支給される。

1日あたりの金額 = 標準報酬**日**額 × **2/3**

#### 4. 出産育児一時金

出産したときに、子供1人につき**50万円**が支給される。

#### 5. 出産手当金

**被保険者が**出産のために会社を休み、給与が支払われない場合に給付される。

予定日の **42日(6週)** 前、出産後 **56日(8週)** までの期間が対象。

1日あたりの金額 = 標準報酬**日**額 × **2/3**

#### 6. 埋葬料

葬儀を行う場合に**5万円**が支給される。
(業務中や勤務中なら労災保険の葬祭料が優先支給)

### 任意継続被保険者

退職時に**2ヶ月以上**の継続した被保険者期間があるとき、
退職日の翌日から**20日以内**に申請すれば、
**最長2年間**、健康保険の被保険者資格を継続できる制度である。
ただし、保険料は**全額自己負担**である。

任意継続のキーワードは「2」

### 国民健康保険

**75歳未満**の、**自営業者**や**未就業者**が加入する。
都道府県、市町村、または国民健康保険組合(特定の職種が対象)が運営する。
**被扶養者という概念がない**ため、配偶者や子供も加入する必要がある。

保険料は市町村によって異なる。

傷病手当金や出産手当金といった**手当金は給付されない**(計算のしようがないからだろう)。

### 後期高齢者医療保険

**75歳以上**(障害認定を受けた人は65歳以上)のすべての人が加入する。
負担割合は原則**1割**である。
年金からの天引きとなり、**市町村(特別区)** が徴収する。
**被扶養者の概念はない**ので全員加入。

### 公的介護保険

将来に介護を受けるための保険。**市町村(特別区)** が運営する。
1ヶ月のサービス利用料が、所得区分に応じて定められた上限額を超えた場合には、
**高額介護サービス費**として超過分が支給される。
要支援1-2、要介護1-5の7段階の要介護認定がある。
ケアプランは本人が作成してもOK。

- **65歳以上**は第一号被保険者
  - 年金が**年額18万円以上**の人は、年金から天引きされる
  - 要介護・要支援になると**無条件で給付**対象
  - 原則**1割**負担、**所得が多いと2-3割**負担
- **40歳-64歳**は第二号被保険者
  - 保険料は医療保険に上乗せ (協会けんぽの保険料は**全国一律**)
  - **末期がん**や認知症といった**特定疾病**によって要介護・要支援になると給付対象
  - 負担は**1割**で固定

### 労災保険

業務災害や通勤災害(寄り道は除く)に給付される保険。
**パートやアルバイトも含む、全ての労働者が対象**である。
役員やフリーランスは対象外だが、中小事業主や一定要件を満たすフリーランスは**特別加入**できる。
保険者は国で、窓口は労基。
保険料は**全額事業主負担**である。

- 療養補償給付
  - 無料で治療を受けられる
  - 現物支給的な扱い
- 休業補償給付
  - **休業4日目** (連続でなくても良い)から、給付基礎日額の**6割相当額**と、別途特別支給金が2割、支給される
- 傷病補償年金
  - **1年6ヶ月**過ぎても治らず、傷病等級1級から3級に該当する場合に、休業補償給付に代わって支給される
- 障害補償給付
  - 障害が残ったときに、程度に応じて年金または一時金が支給される
- 介護補償給付
  - 障害補償年金や傷病補償年金を受けている人が介護サービスを利用した時
- 遺族補償給付
  - 死亡したときに、**遺族の数に応じて年金が支給される**
  - 年金を受け取る家族がいない場合は一時金が支給される
- 葬祭料
  - 葬儀を行う場合に支給 (業務外の場合は健康保険の埋葬料が使える)
- 二次健康診断等給付
  - 健康診断等で業務上の疾病が疑われる場合に、無料で二次診断を受けられる

### 雇用保険

失業や教育訓練に対して給付される保険。
保険者は国で、窓口はハローワークである。
以下に挙げた部分(労働者が利益を得る部分)の保険料は、**事業主と労働者の両方が負担**する。

雇用保険の要件として、以下の月数がよく出てくる。

- 2年間に12ヶ月以上加入
  - 自己都合退職による基本給付
  - 育児休業給付
- 1年間に6ヶ月以上
  - 倒産による基本給付
  - 高年齢雇用継続給付

#### 基本手当

**65歳未満**の求職活動中の人に支給される。
離職日以前の**2年間に被保険者期間が通算12ヶ月以上**(倒産なら1年間に6ヶ月以上)がある人が対象。
離職前6ヶ月の賃金日額の45%-80%が支給される。

- 自己都合退職・定年退職
  - 1年未満は**0日**
  - 1年以上10年未満は**90日**
  - 10年以上20年未満は**120日**
  - 20年以上は**150日**
- 倒産・解雇
  - 6ヶ月以上1年未満は**90日**
  - 最短は、30歳未満 1-5年 **90日**
  - 最長は、45歳以上60歳未満 20年以上 **330日**

自己都合退職の場合、求職の申込後**7日間は待機期間**となり、その後**1ヶ月間は給付制限**がある。
ただし、教育訓練を受けた場合はすぐに受給できる。
受給期間は**離職の翌日から原則1年間**、理由がある場合は**最長4年間**である。

#### 高年齢求職者給付金

65歳以上の人が離職し、保険期間が離職前1年間に6ヶ月以上ある場合に、
基本手当日額の30日分または50日分が支給される。
基本手当はないので、そのかわり。

#### 就職促進給付

基本手当の受給中に、就職が決まった人に支給される。

#### 教育訓練給付

指定の講座を受講・修了した人に、費用の一部が支給される。
保険期間が**3年以上**ある人が対象である。

- 一般教育訓練給付金
  - 費用の**20% (上限10万円)** が支給される
- 特定一般教育訓練給付金
  - 業務独占資格などの講座
  - 費用の**40% (上限20万円)** が支給される

#### 育児休業給付

原則1歳未満、最長2歳未満の子を育てるために休業する人に給付される。
休業開始日前**2年間**に、みなし保険期間(11日以上勤務した月)が通算**12ヶ月以上**である場合に、
賃金日額の**67%**(181日目からは**50%**)が支給される。
ただし、賃金が80%を超えている期間については支給されない。

#### 介護休業給付

**通算93日**まで、**3回まで分けて**取得できる介護休業期間中、
賃金の **67%** 相当額が支給される。

#### 高年齢雇用継続給付

60歳の時より賃金が下がった場合に支給される。

- 期間は65歳未満まで
- 被保険者期間が**5年以上**であること
- 60歳時点の給与の**75%未満**であること
- 給付額は月額の **10%** 相当額
- 2つの種類がある
  - 高年齢雇用継続基本給付金
    - 60歳以降も継続して雇用される人
  - 高年齢再就職給付金
    - 雇用保険の基本手当を100日以上残して再就職した人

## 公的年金の基本

**20歳以上60歳未満**のすべての人に加入義務があるのが**国民年金**である。
会社員などは、さらに**厚生年金**に加入する。
つまり、日本の公的年金は2階建てである。

**国民年金**の被保険者は、以下の3種類に分けられる。

- 第1号被保険者
  - 自営業や学生
  - 20歳以上60歳未満
- 第2号被保険者
  - 会社員や公務員
  - 年齢要件なし
  - 給料天引きで**労使折半**
- 第3号被保険者
  - 第2号被保険者の配偶者
  - 20歳以上60歳未満

以下のとおり、出産育児関連で減免がある。
これらの期間は**納付したものとして扱われる**。

- 第1号被保険者
  - 出産予定日の**前月から4ヶ月間**、保険料が免除される
- 第2号被保険者
  - 産休中・育休中(子が**3歳**になるまで)は、保険料が免除される
  - 事業主分も免除になる

加入義務がなくても任意で加入する人を**任意加入被保険者**という。
なお、満額を得るためなら65歳まで、受給資格を満たすためなら70歳まで、任意加入できる。

**第1号被保険者**に限り、免除・減免・猶予の制度がある。

- 法定免除
  - 障害・生活保護などは**全額免除**になる
- 申請免除
  - 所得が少ない場合、**全額、3/4、半額、1/4**の免除になるかも
  - 本人と世帯主と配偶者の所得で判定
- 学生納付特例制度
  - 20歳以上の学生で収入が少ない場合、猶予されるかも
  - 本人の所得で判定
- 納付猶予制度
  - **50歳未満**で所得が少ない場合、猶予されるかも
  - 本人と配偶者の所得で判定

**厚生年金**の保険料は、標準報酬月額に18.3%を乗じて計算し、労使折半である。
厚生年金は**70歳**になると脱退する。

### 公的年金の給付

公的年金の給付には以下の3つのパターンがある。

- **老齢給付**
- **障害給付**
- **遺族給付**

受給は、市町村等に**裁定請求**することで開始される。
**65歳**になった翌月から、**偶数月**の**15日**に、支給日の前月までの**2ヶ月分**を受け取る。

所得が一定以下の人は**年金生活者支援給付金**を受け取れる場合がある。
消費税が原資であることになっている。(クソ政府が嘘つきがやがって)

年金法における「子」の定義は常に以下のとおり。

- **18歳**到達年度の末日までの未婚子
- **20歳未満**の障害1級2級の未婚子

### 老齢給付(老齢基礎年金)

**老齢基礎年金**は、受給資格期間が**10年以上**ある場合に、65歳から受け取れる。
受給資格期間とは、以下の合計のこと。

- **保険料納付済期間**
  - 産前産後の免除も含む
- **保険料免除期間**
  - 法定免除、申請免除、学生納付特例制度、納付猶予制度
- **合算対象期間**
  - 海外にいたので任意加入だった期間など
  - 受給資格の判定には算入され、年金額の判定には参入されない

年金額は受給資格期間によって変わる。
**480月**(40年✕12ヶ月)なら満額もらえる。

老齢基礎年金額は毎年改定され、2025年度は831,700円。

保険料の計算式は以下のとおり。

保険料 = 831,700円 ✕ (保険料納付済月数 + 保険料免除期間に係数をかけた月数) / 480月

年金は**繰上げ**と**繰下げ**ができる。
支給開始後は、減額・増額された金額が一生涯支給される。

- 繰上げ
  - 60歳まで早められる
  - 一月あたり **0.4%** が減額
- 繰下げ
  - 75歳まで遅らせられる
  - 一月あがり **0.7%** が増額

**付加年金**とは、**第1号被保険者等**の年金上乗せ制度である。
**月額400円**を上乗せして払うと、**納付月数 ✕ 200円**が年金の年額にプラスされる。
たった2年でもとが取れるのでお得。

### 老齢給付(老齢厚生年金)

公的年金の受給資格があり、厚生年金に1ヶ月以上加入していた人が受給できる、2階部分。

厚生年金に1年以上加入している人は、生年月日によっては**特別支給**を受けられる。
65歳よりも前に支給を受けられる制度。男は2025年、女は2030年で廃止される。
**報酬比例部分と定額部分**がある。

65歳になると、それらは**老齢厚生年金と老齢基礎年金**に切り替わる。
このとき、老齢基礎年金が「特別支給における定額部分」を下回らないように、
**経過的加算**なるものを加算し、同じ金額になるよう調整される場合がある。

**加給年金**とは、上乗せ分のこと。
受給要件は、**加入期間が20年以上**であること、子または**65歳未満**の配偶者がいること。
なお、配偶者が65歳を超えると打ち切られ、以降は配偶者の年金に**振替加算**がつく

老齢厚生年金も、老齢基礎年金と同じく繰上げ、繰下げが可能で、増額・減額の割合も同じ。
**繰上げは基礎年金と同時**にする必要がある。
繰下げば別でもOKで、**66歳以降**に請求できる。

**在職老齢年金**は、60歳以降も働きながらもらう老齢**厚生**年金のこと。
報酬と年金の合計が**51万円**を超えると減額される。
厚生年金は70歳までなので、それ以降は保険料負担は免除される。

離婚時には年金を分割できる。
**離婚の翌日から2年間が請求期限**である。

- 合意分割
  - お互いの合意で割合を決める
- 3号分割
  - 国民年金3号被保険者の請求により、合意がなくても1/2にできる

### 障害給付(障害基礎年金)

- 受給要件
  - **障害認定日** (症状固定日 or 1年6ヶ月経過日)に、障害等級が1級か2級であること
  - **保険料納付済期間+保険料免除期間が2/3以上**であるか、直近1年間に滞納がないこと
- 計算式
  - 1級: 老齢基礎年金の満額 ✕ **1.25倍** + 子の加算額
  - 2級: 老齢基礎年金の満額 + 子の加算額

### 障害給付(障害厚生年金)

- 受給要件
  - **障害認定日**に障害等級が1-3級であるか、一定の障害があること
    - 3級より軽度なら**一時金**の一発で修了
  - 納付要件は障害基礎年金と同じ
- 計算式
  - 1級: 報酬比例部分の年金額 ✕ **1.25倍** + 配偶者加給年金
  - 2級: 報酬比例部分の年金額 + 配偶者加給年金
  - 3級: 報酬比例部分の年金額
    - 上記に加え、老齢基礎年金の満額 ✕ **3/4** の最低保証がある。これは、3級には障害基礎年金がなく、老齢基礎年金の加入期間が短い場合に給付額が極端に低くなることがあるため、それを防ぐ目的で設けられているもの。
  - 障害手当金(一時金): 報酬比例部分の年金額 ✕ **2倍**

### 遺族給付 (遺族基礎年金)

- **遺族基礎年金**の受給要件
  - まだ働いてた人: **保険料納付済期間+保険料免除期間が2/3以上**であるか、直近1年間に滞納がないこと
  - もう年金もらってた人: 受給資格期間が**25年以上**あること
  - **子**または**子のある配偶者**であること
- 計算式
  - 老齢基礎年金の満額 + 子の加算額

また、国民年金の**第1号保険者独自**の給付として以下の2つがある。
夫が年金を受け取らずに死亡した場合に利用できる。
どちらか一方のみ選択可能。

- **寡婦年金**の受給資格
  - 妻である(受給期間は**60歳以上65歳未満**の間のみ)
  - 婚姻期間と納付期間が**10年以上**ある
- **死亡一時金**の受給資格
  - 納付期間が**3年以上**であること
  - 遺族が遺族基礎年金を受け取れないこと (子がいない妻を想定)

### 遺族給付 (遺族厚生年金)

- **遺族厚生年金**の受給要件
  - 子の有無は関係ない
  - それ以外は遺族基礎年金と同じ
- 給付対象の優先順位
  - 1. 夫(**55歳以上**)、妻(年齢要件なし)、子
  - 2. 父母(**55歳以上**)
  - 3. 孫
  - 4. 祖父母(**55歳以上**)
- 計算式
  - 老齢厚生年金の報酬比例部分 ✕ **3/4** (ただし最低でも300月の被保険者期間があるものとして計算)

子のある妻は、子供が18歳になると遺族基礎年金が打ち切られる。
また、子がない場合はそもそも遺族基礎年金が受け取れない。
こういった状況の救済として、遺族厚生年金には**中高齢寡婦加算**がある。
なお、65歳になったときに額が減らないように調整する
**経過的寡婦加算**という仕組みもあるが、昔の人だけが対象。

### 併給調整

**1人1年金**が原則であるため、複数ある場合は**いずれか1つを選択**する必要がある。

「基礎+厚生」の組み合わせにおいて、原則として違う種類の年金は併給できないが、以下は認められている。
ただし、いずれも65歳以降であることが条件。

- 障害基礎年金は、すべての厚生年金と組み合わせ可能
- 老齢基礎年金は、遺族厚生年金と組み合わせ可能

特殊な例として、遺族**厚生**年金と老齢**厚生**年金は、厚生年金同士でも併給ができる。
この場合は、以下の多いほうと自分の老齢年金の差額が、
自分の厚生年金に**差額支給**される。

- (配偶者の)遺族厚生年金
- (配偶者の)遺族厚生年金 ✕ 2/3 + (自分の)老齢厚生年金 ✕ 1/2

**雇用保険の基本手当**と、厚生年金の特別支給は併用できず、特別支給が**全額支給停止**になる。
また、**雇用保険の高年齢雇用継続給付**を受ける場合は、厚生年金の**在職老齢年金の一部**が支給停止になる。

**障害厚生年金**と**障害補償年金(労災保険)**を受け取る場合、**後者は減額される**。

### 年金と税金

本人や生計同一親族の保険料を払うと、**全額が社会保険料控除の対象**となる。

老齢(基礎|厚生)年金を受け取った場合は、公的年金控除を差し引いた後、**雑所得**となる。
ただし、障害給付・遺族給付は非課税である。

## 企業・個人事業主の年金

### 企業年金の種類

企業によっては、任意の年金制度として**企業年金**を設けている。
公的年金(国民年金・厚生年金)を補うための私的年金である。

- **確定給付型**: 将来に受け取る給付額まで確定している
- **確定拠出型**: いま払う拠出額(掛金)しか確定していない

### 確定給付年金(DB)

いまは確定給付企業年金というものが残っている。
基金型と規約型がある。
掛け金は**生命保険料控除の対象**となる。

### 確定拠出年金(DC)

**10年以上**加入期間があれば**60歳**から老齢給付金を受給できる。
遅くとも**75歳**までに受給を開始しなくてはならない。

退職等により、もはや掛金は拠出しないけど運用だけ行なっている人を**運用指図者**という。

- **企業型**
  - **最長70歳未満**の厚生年金被保険者が加入できる
  - 掛け金は原則事業主負担
    - 規約を定め、加入者が事業主の掛け金に上乗せできる仕組みを**マッチング拠出**という
  - **月55,000円**が限度
- **個人型(iDeCo)**
  - **最長65歳未満**の国民年金被保険者が加入できる
    - 1号、2号、3号、任意継続いずれもOK
    - ただし2号でマッチング拠出している人は不可
  - 掛金は加入者負担
  - 限度額
    - 1号: **月68,000円** (ただし国民年金基金と枠は共有)
    - 2号: **月23,000円** (DBや企業型DCに入っているか、公務員なら、**月20,000円**)
    - 3号: **月23,000円**

DCの給付には以下の3種類がある。

- 老齢給付金
- 障害給付金
- 死亡一時金

DCの掛金は**全額が小規模企業共済等掛金控除**の対象となる。
老齢給付金を**一時金**として受け取った場合は、**退職所得**となり、退職所得控除の対象となる。
老齢給付金を**年金**として受け取った場合は、**雑所得**となり、公的年金等控除の対象となる。

**中小企業退職金共済制度(中退共)**とは、中小企業向けの社外積立型の退職金制度。
原則全員加入で、掛金は全額事業主負担である。国の補助がある。
会社・個人を問わず、**全額損金にできる**。給付は一時払いとなる。

- 掛金
  - 新規: 掛金の**1/2(上限5,000円)**を1年間助成
  - 増額: 増額分の1/3を1年間助成

### 自営業者の年金制度

自営業には厚生年金や退職金がないので、それらを補うための仕組みがいくつかある。

- 付加年金
  - 詳細は別記
- 国民年金基金
  - iDeCoと合算して**月68,000円**まで
- 小規模企業共済
  - 小規模企業の役員や、個人事業主のための**退職金**制度
  - **月70,000円**まで
  - **全額が小規模企業共済等掛金控除**の対象
  - DCと同じく、一括受け取りなら退職所得、分割なら雑所得

## 中小法人の資金計画

投資家から直接お金を到達するのが**直接金融**である。

- 株式の発行
  - 株主割当増資
    - 新規発行する株式を、既存株主の持ち株比率に応じて割り当てる
  - 第三者割当増資
    - 新規発行する株式を、特定の第三者に割り当てる
  - 公募増資
    - 新規発行する株式を、広く一般の不特定多数に売る
- 債券の発行
  - 公募債
    - 証券会社を通じて広く一般に売る
  - 私募債
    - 機関投資家などと直接取り引きして売る
    - 一定の条件を満たす私募債を保証する特定社債補償制度がある

金融機関から調達するのが**間接金融**である。

- 証書借入: 借用証書を書いて借りる
- 手形借入: 約束手形を振り出して借りる
- 当座借越: 残高を超えて借りる
- ABL / Asset-based Lending: 担保を入れて借りる
- インパクトローン: 外貨で使途制限のないお金を借りる
- 信用保証協会保証付融資: 公的保証により担保と保証金なしで借りる

### 企業の財務状況の把握

損益計算書の構成要素

- 売上高
- 売上原価
  - (売上総利益)
- 販売費および一般管理費
  - (営業利益)
- 営業外収益・費用
  - (経常利益)
- 特別利益・損失
  - (税引前当期純利益)
- 法人税等
  - (当期純利益)

**変動費**は売上に比例してかかるコスト。
**固定費**は売上と無関係なコスト。
売上高から変動費を引いたものを**限界利益**という。
賞味の利益と言えるだろう。
`限界利益 / 売上高`を**限界利益率**という。
`固定費 / 限界利益率`で**損益分岐売上高**を求められる。

### 財務分析

- 売上高総利益率 = 売上総利益 / 売上高 (粗利率)
- 売上高営業利益率 = 営業利益 / 売上高 (販管費まで含めた利益率)
- 売上高経常利益率 = 経常利益 / 売上高 (投資等まで含めた利益率)
- 流動比率 = 流動資産 / 流動負債 (短期的な支払能力)
- 当座比率 = 当座資産 / 流動負債 (短期的な支払能力)
- 固定比率 = 固定資産 / 自己資本 (長期的な安全性)
- 自己資本比率 = 自己資本 / 総資本 (財務の安定性)
- 総資本回転率 = 売上高 / 総資本 (総資本の利用効率)

## ローンとカード

クレジットカードの支払い方法

- 一括払い
  - **ボーナス一括払い**もある
- 分割払い
  - 3回以上は金利がかかる
- リボ払い

クレジットカードのキャッシングには**総量規制**があり、
**年収の1/3を超えてはいけない**と法律に定めがある。
