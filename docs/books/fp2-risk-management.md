# FP2級 / リスク管理

## 保険の基本

リスク管理とは、リスク発生時の損害を回避したり、軽減したり、最小に抑えたりすること。

保険には**公的保険**(社会保険)と**私的保険**(民間保険)がある。

私的保険には3つの分野がある

- 第一分野: **生命保険** / 人の生命や生活の危機に備える
- 第二分野: **損害保険** / 事故の損害に備える
- 第三分野: どちらにも属さない病気や障害に備える

**大数の法則**とは、少数では法則性が見つからなくても、大数で見ると一定の法則が見えること。

**収支相等の原則**とは、保険会社の収入と支出が均衡する方に保険料を算定すること。

クーリング・オフ制度では、契約日かクーリングオフ書面受取日の**遅い方**から**8日以内**に、
**書面または電磁的記録**によって行なうことができる。
保険期間が**1年以下**のものは対象外。

保険会社は保護機構への加入が義務付けられている。
ただし少額短期保険事業者や共済は加入しない。

- 生命保険契約者保護機構
  - **責任準備金等**の**90%**
- 損害保険契約者保護機構
  - 自賠責、地震: **100%**を補償
  - 自動車、火災、短期傷害、海外旅行傷害など: 保険金は、破綻後**3ヶ月は100%**、それ以降は**80%**

**少額短期保険**は、期間は生命1年、損害2年と短い。
相続税は非課税。保険金額は上限1000万円。
保護機構の対象外。返戻金もない。

**ソーベル・マージン比率**とは、**支払能力**を測る指標。
**200%**を下回ると金融庁の是正措置対象になる。

保険の一般的なルールは、分野ごとに**保険法**に定められている。

- 第一分野: 生命保険
  - 告知義務 - 聞かれたら答えるでOK
  - 被保険者の同意 - 被保険者の同意がなければ無効
  - 保険金受取人の変更 - 被保険者の同意がなければ原則無効
- 第2分野: 損害保険
  - 訴求保険 - 事故を知っていたのに申し込んだら無効
- その他共通
  - 請求の権利は3年行使しないと消える

**保険業法**には、保険会社等が健全な運営を行なうためのルールが定められている。
嘘の告知を勧めたり、保険料を便宜したり、将来の不確実な利益を確約したりしたらだめよ。

## 生命保険のしくみと契約

用語

- 保険金: 死亡・高度障害時に受取人に支払われるお金
- 給付金: 手術や入院などのときに受取人に支払われるお金
- 主契約: 保険の主たる契約部分
- 特約: 主契約に付加するオプション

生命保険の種類

- 死亡保険 - 保険期間中に死亡したり高度障害になったら支払い
- 生存保険 - 保険期間終了時に生存していたら支払い
- 生死混合保険 - 上記2つの組み合わせ

保険料は以下3つの**予定基礎率**を用いて算出される。

- 予定死亡率 - 統計的な死亡率。高いなら保険料は上がる。
- 予定利率 - 予想される運用利回り。高いなら保険料は下がる。
- 予定事業費率 - 予想される事業費率。高いと保険料は上がる。

予定死亡率と予定利率から算出するのが**純保険料**で、これは保険金の支払いに充てられる。
予定事業費率から算出するのが**付加保険料**で、これは事業経費に充てられる。
この2つを合わせたものが保険料である。

**剰余金**には以下の3つがある。

- 死差益: 予定死亡率よりも実際の死亡者が少なかった
- 利差益: 予定利率よりも運用益が多かった
- 費差益: 予定事業費率よりも事業費が少なかった

剰余金を財源として契約者に払い戻されるのが**契約者配当金**である。

配当金を支払うかどうかで、保険を分類できる。
配当があると保険料はあがるのが一般的。

- **有配当保険**: 配当金を支払う保険
- **準有配当保険(利差配当つき保険)**: 利差益だけを配当金として支払う
- **無配当保険**: 配当金のない保険

契約者等は保険会社に健康等に関する**事実を告げる義務**がある。
ただし質問に答えるだけでよく、自発的に答える必要はない。
これを**告知義務**という。
告知義務違反があった場合は保険会社は契約を解除できるが、
契約から5年が経過していたり、保険募集人が**告知を妨げた**などの理由があると、解除できない。

以下の3つが揃うと保障が開始される。この日を**責任開始日**という。

- 保険の申し込みが済んでいる
- **告知**または医師の診察が完了
- **初回の保険料の払込み**が完了

保険料の支払いにはいくつかの方法がある。

- 月払い
- 半年払い
- 年払い
- 一時払い: 期間全体の保険料を契約時に払う
- 全期前納: 期間全体の保険料を保険会社に預けて、年1回保険料に充当していく

契約日と同じ日を**契約応当日**という。
支払いを忘れてもすぐに失効しないように**払込猶予期間**が設定されている。

- 月払いなら、払込期月の翌月**1日から末日**まで
- 年払い・半月払いなら、払込期月の翌月**1日から翌々月の契約応当日**まで

**自動振替貸付制度**は、払込猶予期間を過ぎても支払いがない場合に、
解約返戻金を限度に保険会社が保険料を**立て替え**る制度。
立替した保険料は生命保険料控除の対象となる。

**契約者貸付制度**は、解約返戻金の一定範囲内で保険会社から**貸付を受ける**制度。
利息が発生する。

一度失効した保険でも、一定期間内に手続きをしたうえ未払いの保険料を支払えば、
保険を**復活**させられる。このとき、保険料は変わらない。

**必要保障額**とは、残された遺族が一定期間生活するのに必要な金額のこと。

必要保障額 = 将来の生活費 + その他必要資金 - (公的年金 + 配偶者収入 + 保有金融資産)

## 生命保険の種類と契約

### 定期保険

**逓増**とは、徐々に増えること。**逓減**は、徐々に減ること。

定期保険は、死亡や高度障害を**一定期間**保障する保険。
**満期保険金はない**、**掛け捨て**、**保険料が安い**のが特徴。

- 平準定期保険
  - 保険金額も保険料も一定。
- 逓増定期保険
  - **保険金額が段階的に増加**する。保険料は一定。
- 逓減定期保険
  - **保険金額が段階的に減少**する。一般に、保険料は一定。
- 収入保障保険
  - 死亡保険金等を**年金形式**で受け取れる
  - 年金ではなく一時金でも受け取れるが**受取金額は減る**

### 終身保険

終身保険は、死亡や高度障害を**一生涯**保障する保険。
貯蓄性も併せ持つが、早期解約すると損になる。

- 終身払い
  - 契約中は**一生涯支払いが続く**
- 有期払い
  - **一定期間で支払いが終了**する
  - 保障は一生涯続く

終身保険にはいくつかの種類がある。

- 終身保険
  - 保障は一生涯
  - 解約返戻金を年金方式で受取ることも可能
  - **死亡保険金は相続税対策として活用しやすい**
- 低解約返戻金型終身保険
  - **解約返戻金が抑えられている**ため、保険料が安い
  - 払込期間が終了すると返戻金は通常と同程度になる
- 定期保険特約付き終身保険
  - 主契約となる終身保険に、定期保険をトッピングして一定期間保障を厚くする
  - 全期型と更新型(自動更新&告知不要)がある
- 積立利率変動型終身保険
  - 市場金利を積立利率に定期的に反映するため、場合によっては増える
  - 保険金や解約返戻金には最低保証がある
- 利率変動型積立終身保険(アカウント型保険)
  - 保険料が保障部分と積立部分に振り分けられる
  - 積立部分と保障部分は自由に組み替えられる
  - 払込満了時に、積立部分が終身保険に移行する
