# FP2級 / リスク管理

## 保険の基本

リスク管理とは、リスク発生時の損害を回避したり、軽減したり、最小に抑えたりすること。

保険には**公的保険**(社会保険)と**私的保険**(民間保険)がある。

私的保険には3つの分野がある

- 第一分野: **生命保険** / 人の生命や生活の危機に備える
- 第二分野: **損害保険** / 事故の損害に備える
- 第三分野: どちらにも属さない病気や障害に備える

**大数の法則**とは、少数では法則性が見つからなくても、大数で見ると一定の法則が見えること。

**収支相等の原則**とは、保険会社の収入と支出が均衡する方に保険料を算定すること。

クーリング・オフ制度では、契約日かクーリングオフ書面受取日の**遅い方**から**8日以内**に、
**書面または電磁的記録**によって行なうことができる。
保険期間が**1年以下**のものは対象外。

保険会社は保護機構への加入が義務付けられている。
ただし少額短期保険事業者や共済は加入しない。

- 生命保険契約者保護機構
  - **責任準備金等**の**90%**
- 損害保険契約者保護機構
  - 自賠責、地震: **100%**を補償
  - 自動車、火災、短期傷害、海外旅行傷害など: 保険金は、破綻後**3ヶ月は100%**、それ以降は**80%**

**少額短期保険**は、期間は生命1年、損害2年と短い。
相続税は非課税。保険金額は上限1000万円。
保護機構の対象外。返戻金もない。

**ソーベル・マージン比率**とは、**支払能力**を測る指標。
**200%**を下回ると金融庁の是正措置対象になる。

保険の一般的なルールは、分野ごとに**保険法**に定められている。

- 第一分野: 生命保険
  - 告知義務 - 聞かれたら答えるでOK
  - 被保険者の同意 - 被保険者の同意がなければ無効
  - 保険金受取人の変更 - 被保険者の同意がなければ原則無効
- 第2分野: 損害保険
  - 訴求保険 - 事故を知っていたのに申し込んだら無効
- その他共通
  - 請求の権利は3年行使しないと消える

**保険業法**には、保険会社等が健全な運営を行なうためのルールが定められている。
嘘の告知を勧めたり、保険料を便宜したり、将来の不確実な利益を確約したりしたらだめよ。
