# FP2級 / 金融資産運用

## 経済・金融市場の基礎

### 経済指標・景気指標

- GDP: **国内**の経済活動で生み出された付加価値
- 三面等価の原則: 生産(付加価値)=支出(消費)=分配(給与)

**景気動向指数**にはいくつかの種類がある。

- CI 景気変動の大きさやテンポ (主流)
- DI 景気波及の度合い

CIとDIにはそれぞれ以下がある。

- 先行指数
  - 新規求人数(これはフロー): これから売れそうだから新たに人を集めよう
  - 新設住宅着工床面積: 先行き明るそうだしデカイもの買おう
  - 株価: あらゆる材料に超敏感
- 一致指数 (**景気判断**に使うのはコレ)
  - 生産指数: いまどれくらい作ってるか
  - 有効求人倍率(これはストック): いまの雇用情勢はどうか
- 遅行指数
  - 完全失業率: 頑張ったけどもうだめだクビにしよう
  - 物価: 給料減らされたし切り詰めよう
  - 法人税収入: 頑張ったけど利益が上がらなかった

**日銀短観**は**年4回**発表される。
業況判断DIが最も有名。
**良い**と答えた企業の割合と**悪い**と答えた企業の割合の差。

**マネーストック**は国や金融機関**以外**の経済主体が有する**通貨の総量**のこと。

物価指数には、**総務省**が発表する消費者物価指数と、
**日本銀行**が毎月発表する企業物価指数がある。

### 金融市場と金融政策

金融とは「お金を融通(貸し借り)する」こと。
金融市場はいくつかの分類がある。

- **短期金融市場**: 取引期間が**1年未満**
  - **インターバンク市場**
    - **金融機関(銀行、証券会社、保険会社)のみ**が参加可能
    - 手形市場、コール市場(無担保コール翌日物で有名)などがある
  - オープン市場
    - 一般企業も参加可能
- **長期金融市場**: 取引期間が**1年以上**
  - 株式市場
  - 債券市場
    - 債券は取引所を介さずに証券会社と直接**店頭取引・相対取引**する

## 金融資産・顧客の保護と法律

### 金融機関等の破綻と顧客保護

**銀行**などが破綻したときに預金者を保護するのが**預金保険制度**である。

- 保護対象
  - 預金、定期積金、元本補てんありの金銭信託など
    - **元本1000万円と利息**が保護
  - 決済用預金
    - 当座預金・振替貯金のこと
    - **全額**が保護
    - **無利息、要求払い、決済サービス**
    - 「決済口座(振り込みや引き落としができる)」とは別物なので注意
- 保護対象外
  - 外貨預金
  - 譲渡性預金
  - 元本補てんのない金銭信託

複数口座があるときは名寄せされる。
金融機関が合併したときは**元本1000万円✕金融機関数**が1年間に限り保護される。

**証券会社**が破綻したときに投資家を保護するのが**日本投資者保護基金**である。
証券会社が**分別管理**を怠っていた場合に、1人最大**1000万円まで**を補償する。

### 金融商品の関連法規

**金融サービス提供法**は投資家(法人含む)を保護するための法律。
**重要事項等の説明**や**損害賠償責任**を規定している。
預貯金、保険、有価証券などが対象で、ゴルフ会員権や金地金などは対象外。

**消費者契約法**は消費者(**個人**)を保護するための法律。

**Fiduciary Duty / フィデューシャリー・デューティー**とは、
受託者（fiduciary）が委託者の利益を最優先に行動しなければならない義務のこと。
方針策定、状況開示、見直しをする必要がある。

**金融商品取引法**は、金融商品の販売等におけるルールを定めた、投資家保護のための法律。
特定投資家(プロ)と一般投資家(アマチュア)にそれぞれ異なる規制をかけている。
**適合性の原則**とは、顧客の実情に合った商品等を提供しなければならない原則のこと。

**金融ADR機関制度**とは、金融機関と利用者の間のトラブルを、
裁判によらず、**指定紛争解決機関**(各種協会とそこに所属する弁護士)で仲裁すること。

**犯罪収益移転防止法**では、犯罪収益をマネロンすることなどを防止するため、
本人確認と7年の書類保存が義務付けられている。
金融機関のほか貴金属買取業者なども該当する。

## 金融商品の種類

### 金融商品の分類

分類の基準は3つ。一般に、収益性と安全性は相反する。

- 安全性
  - 元本保証
  - 利息等支払いの確実性
- 流動性
  - 運用期間
  - 払い戻しができない期間(据置期間)
  - 解約手数料
  - 換金手続きの煩雑さや、現金が振り込まれるまでの時間
- 収益性
  - キャピタルゲインとインカムゲイン

### 貯蓄型金融商品

**貯蓄型金融商品**とは、元本保証があり、いつでも引き出せる預貯金のこと。

年平均利回り = (収益 / 預入年数) / 元本

利息の計算には種類がある。

- 単利 = 元本 ✕ (1+利率✕年数)
- 1年複利 = 元本 ✕ (1+利率)^年数
- 半年複利 = 元本 ✕ (1+利率/2)^年数✕2

利息は**20.315%**で**源泉分離課税**され、確定申告は不可。

### 銀行の金融商品

#### 銀行の金融商品

- 流動性預金 (変動金利)
  - 普通預金
    - 決済に使える普通の口座
  - 貯蓄預金
    - 一定額以上の残高があれば高めの金利がつく
    - 決済口座(振込、引き落としなど)としては**利用不可**
- 定期性預金 (固定金利)
  - スーパー定期
    - **3年未満は単利型のみ**、以上なら半年複利も可
    - 中途換金には**中途解約利率**が適用
  - 大口定期預金
    - **単利型のみ**
    - 中途換金には**中途解約利率**が適用
    - **マル優は使えない**
      - 障害者等なら元本350万円まで利子が非課税になる制度
    - まとまった資金(**1000万円以上**)があるときは、比較的高めの金利がつくので有利
  - 期日指定定期預金
    - **1年複利**
    - 満期日指定によりペナルティなしで解約可能
    - いつ使うか正確にはわからないが、当面は使わないお金に最適

#### ゆうちょ銀行の金融商品

- 流動性貯金 (変動金利)
  - 通常貯金: 普通預金と同じ
  - 通常貯蓄貯金: 貯蓄預金と同じ
- 定期性貯金 (固定金利)
  - 定額貯金
    - 最長**10年**
    - **半年複利**
    - 半年経てば**ペナルティなし**で解約可能
  - 定期貯金
    - 期間はいくつかから選べる
    - 3年未満は**単利型のみ**、以上は**半年複利型のみ**
    - 中途換金には**中途解約利率**が適用

#### 財形制度

勤労者が利用できる財産形成のための制度。

**一般財形**は、原則3年以上で行なう積立て。開始から1年経てばいつでも引き出し可能。

**財形住宅と財形年金**は、55歳未満が5年以上で行なう積立て。
おおむね、あわせて550万まで。目的外引き出しは遡及課税される。

#### 仕組預金

満期日の繰り上げリスクや元本割れリスクと引き換えに、
高めの金利が設定された定期預金のこと。

## 債券

### 債券の基本

債券とは、国や企業などが資金を調達することを目的として発行する借用証書。

- 国債
- 地方債
- 社債(事業債)
- 金融債: 特定の金融機関が発行

用語

- 償還期限: 満期の時期
- 額面金額: 最低申込単位
- 発行価格: 額面100円あたりで表示した発売価格
- 表面利率: 額面金額に対する利率

発行価格により区分される。

- パー発行: 額面と同じ
- アンダーパー発行: 額面より低い価格で発行
- オーバーパー発行: 額面より高い価格で発行

利払い方式による分類

- 利付債: 定期的に利息が支払われる債券
- 割引債: アンダーパー発行された債券

個人向け国債には3つの種類がある。
いずれも最低金利が**0.05%**確保されており、**半年ごと**に利払いがある。
額面**1万円**から購入可能。
購入から**1年が経過**すると換金できるが、**直近2回分の利子✕0.79685(源泉徴収分を除いた部分)**が差し引かれる。

- 変動10年
  - 基準金利 ✕ **0.66**
- 固定5年
  - 基準金利 - **0.05**
- 固定3年
  - 基準金利 - **0.03**

債権の利回り計算においては、100円あたりに条件を揃えたうえで、
インカムゲインとキャピタルゲインを足し合わせて求める。

- インカムゲイン: 表面利率 ÷ `(発行|購入)価格`
- キャピタルゲイン: 売却時と入手時の差額(単年あたり) ÷ `(発行|購入)価格`

利回りには4つの種類がある。

- 直接利回り
  - 投資に対する**毎年**の利子の割合
  - (これだけはキャピタルゲインは関係ない)
- 応募者利回り
  - **最初から最後まで**保有したときの利回り
- 最終利回り
  - **途中から最後まで**保有したときの利回り
- 所有期間利回り
  - **途中から途中まで**保有したときの利回り

**イールドカーブ**とは、利回りと残存期間で描かれるカーブ。
通常は右肩上がりの順イールドになるが、
急激な金融引締時には右肩下がりの逆イールドになることも。
景気拡大時にはスティープ化し、景気後退時にはフラット化する。

### 債券のリスク

- 価格変動リスク
  - 新発債の金利に合わせて既発債の価格が市場で調整されるため、市場金利と債券価格は**逆方向**に動く
- 信用リスク
  - 格付けが**BBB**以上であれば**投資適格債**、それ未満を**投資不適格債(投機的債券)**という
  - 格付けが高いほど安全で、債券価格は高く、利回りは低くなる
- カントリーリスク
- 流動性リスク
- 為替変動リスク

**デュレーション**とは、キャッシュフローの重心のことで、
「投資した金額がいつ頃返ってくるか」を表す。
残存期間の**長い**債券は、デュレーションは長くなる。
利率が**低い**債券は、デュレーションは長くなる。

割引債は満期まで1円も返ってこないのでデュレーションは最長になる。
利付債なら早期に利子がもらえるので、利率が高いほどデュレーションは短くなる。

## 株式

### 株式の基本

株で得た資金には返済義務がない分、様々な権利がついてくる。

- 経営参加権
- 剰余金配当請求権: 配当もらえる
- 残余財産配分請求権: 潰れたとき

単元株とは売買単位のことで、100株。
単元未満で取引できる仕組みとして以下がある。

- 株式累積投資(通称るいとう): 一定額を積立て
- 株式ミニ投資: 10分の1単元で売買

1株をいくつかに分割することを株式分割という。
`1:X`のように表現される。

東京証券取引所には4つの市場区分がある。

- プライム
- スタンダード
- グロース
- TOKYO PRO Market (プロ向け)

### 株式の取引

信用取引の種類

- 制度信用取引
  - 証券取引所がルールを決める
  - 決済期限が最長**6ヶ月**
- 一般信用取引
  - 証券会社と投資家でルールを決める
  - 決済期限は無期限もあり

信用取引の決済

- 反対売買による差金決済
  - 信用買いした株を売り、差額をお金で決済する
  - 信用売りした株を買い、差額をお金で決済する
- 株式を使った決済
  - 信用買いで借りた資金を返済して、現物株を受け取る -> **現引き**
  - 信用売りで借りた株を返済して、現金を受け取る -> **現渡し**

株式には代表的な指標がある。

- 日経平均株価 / 日経225
  - **修正平均株価**
  - **株価が高い値がさ株**の影響を受けやすい
  - 東証プライムのみ対象
- 東証株価指数 / TOPIX
  - **時価総額加重平均型**
  - **時価総額の大きい大型株**の影響を受けやすい
  - 東証3市場が対象
- 東証プライム市場指数
  - 特徴はTOPIXと同じで、市場はプライムのみ
- JPX日経インデックス400
  - 東証3市場の素性の良い400銘柄
  - 年金機関投資家などが使う

海外の指標

- NYダウ: NY証券取引所およびナスダックのうち、30銘柄を株価平均
- SP500: NY証券取引所およびナスダックのうち、500銘柄を時価総額加重平均
- ナスダック総合指数: ナスダックの全銘柄を時価総額加重平均

個別銘柄の指標

- PER
  - 株価が**1株あたり純利益の何倍か**で、割安割高を表す
  - `株価 / 1株あたり純利益`
- PBR
  - 株価が**1株あたり純資産の何杯か**で、割安割高を表す
  - `株価 / 1株あたり純資産`
- ROE (Return on Equity)
  - 投下資本に対するリターンの大きさで、**効率的に利益を挙げているか**を表す
  - `当期純利益 / 自己資本`
- 自己資本比率
  - **財務の安全性が高いか**を表す
  - `自己資本 / 総資産`
- 配当利回り
  - 株価に対する**配当金の割合**で、割安割高を表す
  - `1株あたりの配当金 / 株価`
- 配当性向
  - 利益をどれくらい**株主に還元しているか**
  - `年間配当金 / 当期純利益`

### 決算短信

決算内容をコンパクトにまとめたレポートのことで、以下のような内容が載っている。

- 売上高、営業利益、経常利益、(1株あたり)当期純利益
- 自己資本当期純利益率: ROEのこと
- 総資産経常利益率: 事業の効率性を表す
- 売上高営業利益率: 収益性を表す
- 連結業績予想: 来期の予想業績でみんなが注目している
