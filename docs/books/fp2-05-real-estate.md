# FP2級 / 不動産

## 不動産の基本

### 土地の価格

- 公示価格
  - **1月1日** 基準、国交省
- 基準値標準価格(都道府県地価調査)
  - **7月1日**基準、都道府県、公示価格の100%
- 相続税路線価
  - 1月1日基準、国税庁、公示価格の**80%**
- 固定資産税評価額
  - **1月1日(3年ごとに見直し)**、市区町村or都、公示価格の**70%**

### 鑑定評価

不動産価格の適正な評価は不動産鑑定士によって**鑑定評価**により行われる。
不動産の効用が**最高度**に発揮される「最有効使用」を前提として鑑定する。

- 原価法
  - 再調達価格に**減価修正**を行って求める
- 取引事例比較法
  - 似た条件の事例を参考にし、**時点修正や事情補正、地域要因等の比較**をして求める
- 収益還元法
  - 詳細は[こちら](#不動産投資の収益性)に記載

### 不動産登記

不動産登記とは、土地建物の状況や権利関係を**法務局(登記所)** で記録したもの。

- **表題部**: **表示に関する登記**
  - 所在、地番、地目、地積、家屋番号...
- **権利部**: **権利に関する登記**
  - 甲区
    - **所有権の保存、移転**、差し押さえなど
    - 最初の登記が**所有権保存登記**、以降の登記が**所有権移転登記**
  - 乙区
    - **抵当権**、賃借権等
    - 弁済しても抹消登記しない限り抵当権は消えない

不動産登記は、自信が権利者であることの主張(**対抗力**)に使える。
なお、借地権者が**自己を所有者として登記した建物を所有**している場合や、
賃借人が**建物の引き渡し**を受けている場合は、
登記がなくても対抗力を行使できる。

本登記の要件が整わない場合は**仮登記**をしておくことができる。
あくまで**順位保全**のためにだけ使える。

公示内容どおりに権利を発生させる力を**公信力**という。
登記に公信力はなく、それを信じて誤った取引をしても**法的に保護されない**。

権利部の登記は**任意**である。
ただし、**相続登記は3年以内**(遡及適用あり)の登記が義務付けられている。

**住所氏名の変更登記は2年以内の登記**(遡及適用あり)が義務付けられている。

**登記識別情報**は12桁の英数字で、本人確認書類として使われる。
登記で新たに権利が発生したときに発行され、再発行はできない。
**登記識別情報通知**として申請人に交付される。

登記事項証明書や登記事項要約書は**誰でも交付請求できる**。

不動産登記名義人の名前・住所から、全国の不動産を一括調査して、
一覧として証明書として発行する制度を**所有不動産記録証明制度**という。

不動産の状況を調べるための登記以外の資料として以下がある。

- **公図**
  - 地図っぽいなにかで、**精度は低い**
- **地図(14条地図)**
  - 土地の位置などが記載された地図で、**精度は高い**
  - すべての土地について備えられているわけではない
- 地積測量図
  - 土地の形や面積を示した図で、測量により作成される

資料は法務局か地区町村で得られる。

- 法務局
  - 登記事項証明書、要約書
  - **公図**
  - **地図(14条地図)**
  - 地積測量図
- 市区町村
  - 固定資産課税台帳
  - 都市計画図

## 不動産の取引

### 宅地建物取引業法

以下をビジネスとして行なうには宅建業の免許が必要。

- 売買・交換を**自ら行なう**
- 売買・交換・貸借の**代理をする**
- 売買・交換・貸借の**媒介をする**

宅建業を行なうには従業員**5人に1人**の割合で宅地建物取引士を置く義務がある。
宅地建物取引士の独占業務として、**契約前**の重要事項説明などがある。

媒介契約の種類

- 一般媒介契約
  - 唯一、**複数業者への依頼**が可能
- 専任媒介契約
  - 契約期間は**3ヶ月以内**
  - **2週間に1回**の報告義務
  - **7営業日以内**の物件情報登録義務
- 専属専任媒介契約
  - 唯一、自己発見取引は不可(必ず不動産会社を通しての契約が必要)
  - 契約期間は**3ヶ月以内**
  - **1週間に1回**の報告義務
  - **5営業日以内**の物件情報登録義務

宅建業者の報酬限度額は以下の通り。

- 売買の媒介の場合(通常)
  - 200万円以下の部分: 売買価格 × 5%
  - 400万円以下の部分: 売買価格 × 4%
  - それらを超える部分: 売買価格 × 3%
- 売買の媒介の場合(特例)
  - 800万円以下の「低廉な空き家」は、売り主買い主それぞれ30万円
- 売買の代理の場合
  - 媒介の2倍
- 賃借の場合
  - 貸主借主のどちらも含めて、**家賃1ヶ月分**が上限

### 不動産の取引に関する法律

手付とは、代金の一部。
解約手付とは、契約を解除できる権利を発生させる手付のこと。

契約の履行前であれば、買い主は**手付の放棄**、
売り主は**手付の倍額を返却**すれば契約を解除できる。

**売り主が宅建業者**で買い主が素人なら、手付は**2割**まで。

**物件の引き渡し前**に誰のせいでもない不可抗力で建物が滅失したときは、
買い主は**代金の支払いを拒否できる**。

**契約内容に適合しない**場合は、売り主は**契約不適合責任**を負う。
買主は知った日から**1年以内**であれば、
追完、代金減額、損害賠償、契約解除などを請求できる。
ただし、売主が不適合を知りながら売っていた場合は、1年以上経っていてもOK。
なお、新築の場合は**10年間**は品確法による瑕疵担保責任がある。

**債務不履行**の種類には以下がある。

- 履行遅滞: できるのにやらない
- 不完全履行: やったけど足りてない
- 履行不能: そもそもできない

債務者に帰責事由があるなら、債権者は**損害賠償請求**ができる。
催促しても**期間内に履行がないときは契約解除ができる**。
なお、履行不能なら即時に契約解除できる。

一つの不動産を複数人で持つことを**共有、共有物**という。
それぞれの人の所有権を**持分**という。
持分は**単独で処分できる**が、共有物の**管理行為や変更・処分行為は単独ではできない**。

区分所有建物の床面積は**内法（うちのり）面積**で登記される。
ただし、パンフレット上の専有面積などでは**壁心（へきしん）面積**で盛って表示される。
バルコニーは共有部分なので専有面積には含まれない。

取引における面積の扱いは2つある。
**公募取引**は登記記録ベースで行うもので、実際と違っていても**精算はしない**。
**実測取引**では、差異があれば**精算を行なう**。

未成年者が不動産取引をするには法定代理人が必要。
もし単独で行っても**契約は取り消せる**。

徒歩の距離は1分80mで繰り上げで計算する。

取引態様が「売り主」なら仲介手数料はかからないが「媒介」ならかかる。

## 不動産に関する法令上の規制

### 借地借家法

借地借家法は**借主側の保護**のための法律。

借地権とは、**建物の所有を目的に、土地を借りる権利**のこと。

- 普通借地権
  - **契約の更新**のある、建物の所有のための借地権
  - 契約期間は**30年以上**で定める
  - もし合意がなくても、建物がある場合は**同一条件で契約が更新**される
    - 期間は1回目は**20年以上**、2回目は**10年以上**
  - 貸主が更新を拒絶するには**正当事由**が必要
  - 契約の更新がされない場合は、借主は**建物買取請求権**を行使できる
- 定期借地権
  - **契約の更新がない借地権**のことで、**必ず土地の返還が必要**
  - 一般定期借地権
    - **50年以上**
    - **更地で返還**
    - 契約手法は**書面**であればなんでもOK
  - 事業用定期借地権等
    - **10年以上50年未満**
    - **事業用建物のみ**が対象であり、いかなる居住物件も不可
    - 契約手法は**公正証書**に限る
    - **更地で返還**
  - 建物譲渡特約付借地権
    - 30年以上
    - 建物付きで返還
    - 契約手法の制限は特になし

借家権とは、**建物を借りる権利**のこと。

- 普通借家権
  - **1年以上**で定める。1年以下だと**期間の定めのない契約**とみなされる。
  - 貸主からの更新拒絶には**正当事由**が必要
  - 契約方法に制約はない
- 定期借家権
  - **契約更新のない借家契約**
  - 契約期間に制約はない
  - 契約方法は**書面**
  - 契約期間が**1年以上**の場合、賃貸人は**1年前〜6か月前の間**に通知が必要
  - 事業用途でもOK

**賃借権**の登記がなくても鍵の引渡し等を受けていれば権利を対抗できる。

とりつけたエアコンなどの造作物は、**造作買取請求権**により買取りを請求できる。
ただし、**特約で排除**することも可能。

**家賃の増減額請求権**は貸主借主の両方にある。
普通借家契約においては、一定期間家賃を**減額**しない特約は無効となる。
増額しない特約や、定期借家の場合は制約はない。

**通常の使用**によって生じた**損耗や経年変化**は、賃借人は原状回復義務を負わない。

### 区分所有法

1棟の一部を独立して所有できる建物を区分所有建物という。
その使用や管理のルールを定めたのが区分所有法である。
**占有者(賃借人)も守らなければならない**。

- 専有部分
  - **区分所有の持分は専有部分の床面積の合計**で決まる
- 共用部分
  - **法定共用部分**: エントランス、バルコニーほか
  - **規約共用部分**: 管理人室、集会所など、本来は専有部分だが規約で共有部分とされた部分

専有部分を利用するために必要となる、敷地に関する権利を**敷地利用権**という。
これだけを**分離して処分はできない**。

区分所有者は管理組合を構成し、任意脱退することはできない。
管理組合は**規約**を定める。
**毎年1回は必ず集会を招集**する必要がある。
招集通知は**1週間前**までに行なう。ただし建替決議なら2か月前。
集会では、区分所有者の頭数と、議決権(持分比率)の両方での決議が必要。

- **共用部分の変更で軽微なもの**: **過半数**
- **規約改変**: **4分の3以上**
- **建替え**: **5分の4以上**

### 都市計画法

都市計画法は、住みよい街をつくるための法律。
都市計画区域を定めることにより目的を達成する。

- 線引き区域
  - 市街化区域: **既に市街地 or 10年以内に市街化したい区域**で、**用途地域**が設定される
    - 一低住専、二低住専、田園住居、一中高住専、二中高住専、一住居、二住居、準住居
    - 近接商業、商業
    - 準工業、工業、工業専用
  - 市街化調整区域: **市街化を抑制すべき区域**
- 非線引き区域

以下の要件に当てはまる開発行為は、**都道府県知事等の許可**が必要となる。
ただし、市街地再開発事業や土地区画整理事業の場合は不要。
**工事完了の公告**があるまでは建築物を**建設することはできない**。

- 市街化区域で**1000㎡以上**である
- 市街化調整区域であって、**農林漁業に係る建物等ではない**
- 非線引き区域、準都市計画区域で、**3000㎡以上**である
- 1ha以上である

### 建築基準法

#### 用途制限

用途地域ごとに建築できるものが制限されている。これを**用途制限**という。

- **保育所、診療所**はどこでもOK
- **住宅、図書館、老人ホーム**は、**工業専用地域**以外ならどこでもOK
- **ホテル、旅館**は**一住居から準工業まで**ならOK(住専系は不可)

#### 道路、接道義務、セットバック

建築基準法上の道路とは、幅員**4m以上**の道路のこと。
それ未満の道は**2項道路**と呼ばれる。

建物の敷地は**2m以上**道路に面していないとならない、**接道義務**がある。
2項道路の場合、道路の**中心線から2m下がった線までが道路**
(反対側が川や崖なら、道路の**反対側から4m下がったところまでが道路**)とみなされる。
この後退のことを**セットバック**といい、
この部分には建物を建築できないし、**建ぺい率や容積率の計算で敷地面積に含めない**。

#### 建蔽率

**建蔽率**とは、敷地面積に対する建築面積の割合のこと。
用途地域ごとに定められた建蔽率の上限を**指定建蔽率**という。
以下の場合は緩和される。

- 防火関連で **+10%**
  - **防火地域**にある**耐火建築物** (ただし**建蔽率80%** の地域内ならそもそも制限がなくなり100%となる)
  - **準防火地域**にある**耐火建築物・準耐火建築物**
- **角地**なら **+10%**

#### 容積率

**容積率**とは、敷地面積に対する**延べ面積**の割合のこと。
用途地域ごとに定められた容積率の上限を**指定容積率**という。
接している**最も広い道路**の幅員が**12m未満**である場合、以下の**低い方**が容積率の上限となる。

- 指定容積率
- **前面道路の幅員 ✕ 法定定数(住居系用途地域なら10分の4、その他は10分の6)**

#### 高さ制限

建築物には様々な**高さ制限**がある。
斜線制限では、道路の反対側や隣地の境界線から、一定の斜線を引いた内側に高さを規制する。

空間確保のための制限が2つ。

- 絶対高さ制限
  - **10m**または**12m**までに制限
  - **一低住専、二低住専、田園住居**が対象
- 隣地斜線制限
  - **一低住専、二低住専、田園住居を除く地域**が対象

日当たりに関する制限が2つある。日陰規制が優先される。

- 日影規制
  - 必要に応じて**条例**により高さ制限をかける
  - **商業、工業、工業専用**地域を除いてどこでも規制可能
  - 対象外地域であっても、冬至日に対象地域に日陰を生じさせる場合は対象となる
- 北側斜線制限
  - 一低住専、二低住専、田園住居、一中高住専、二中高住専などの**住宅専用系**が対象

安全のための制限が1つ。

- 道路車線制限
  - すべての地域が対象

### 複数地域にまたがる場合の規制

- 敷地が複数の**用途地域**にまたがる場合は、**過半主義**が適用される。
- 敷地が複数の**防火地域**にまたがる場合は、**厳しい方**が適用される。
- 敷地が複数の**指定建蔽率・指定容積率**の地域にまたがる場合は、**加重平均**で求める。

### 農地法

農地法は農地や農地や採草放牧地の売買、転用を規制する法律。

- 第3条: 権利の移動の制限
  - **農業委員会**が許可する
- 第4条: 転用の制限、第5条: 転用目的での権利移動の制限
  - **都道府県知事**が許可する
  - ただし市街化区域なら同業委員会への届出のみでOK

### 国土利用計画法

国土利用計画法は、土地を総合的・計画的に利用するための法律。

- 規制区域なら**許可制**
- 注視区域、監視区域なら**事前届出制**
- それ以外で一定面積以上なら**事後届出制**

## 不動産にかかる税金と特例

### 不動産取得税

土地建物を取得したときにかかる都道府県税。
購入だけでなく、**贈与**や**増改築**も含む。ただし、**相続**の場合はかからない。

不動産取得税 = 固定資産税評価額(課税標準) ✕ **3%** (住宅以外の建物は4%)

- (建物) 住宅の課税標準の特例
  - **建物**なら評価額を**1200万円**(中古なら年数に応じた額)を控除できる
    - 床面積が**50㎡以上240㎡以下**であること
    - 賃貸用建物でもOK
- (土地) 宅地の取得に係る特例
  - **宅地**なら評価額を**2分の1**にできる
- (土地) 住宅用地の税額軽減の特例
  - **住宅用地**なら税額から(45000円 or 床面積の2倍の面積分の税相当額 の大きい方) を控除
  - だいたい床面積の2倍くらいは土地がいるだろうからそこに対しては課税しないという配慮

### 登録免許税

登録免許税は権利登記の際に課される国税。
不動産取得税と異なり、**相続であっても課税**される。

登録免許税 = (固定資産税評価額 or 債権金額) ✕ 税率

- 土地の税率は**2%**を基本としつつ、**相続の場合に限り0.4%**に軽減される (評価額100万以下なら非課税)
- 建物の税率は**2%**を基本としつつ、**相続・新築・自己居住用・住宅ローン関連では0.4%以下**に軽減される。

所有権移転登記等では**固定資産税評価額**が課税標準となる。
抵当権設定登記では**債権金額**が課税標準となる。

### 消費税

- 土地の譲渡・貸付は非課税。
- 建物の譲渡・貸付は課税。ただし**住宅を除く**。

### 印紙税

文書に課される国税。
金額に応じてかかる。金額がない場合は**200円**。
貼り付けがなくても契約は有効だが**過怠税**が課される。
財産権が移転するわけでなく重要度が低いので、**建物の賃貸借契約書は課税されない**。

### 固定資産税

不動産の保有に対して課される市町村税。
**1月1日の所有者**に課される。
**都市計画税**と合わせて納付する。

税額 = 固定資産税評価額(課税標準) ✕ **1.4%(標準税率)**

居住用住宅には減免措置がある。

- 土地
  - **200㎡以下**の住宅用地は**評価額を6分の1**に
  - **200㎡超**の住宅用地は**評価額を3分の1**に
- 建物
  - 新築住宅なら**120㎡**までの部分の**税額を2分の1**に
    - 一戸建てなら**3年**(長期優良なら+2年)
    - マンションなら**5年**(長期優良なら+2年)

### 都市計画税

公園等の費用のための市町村税。
**市街化区域内**に課される。
固定資産税と異なり税率は変更不可。

税額 = 固定資産税評価額(課税標準) ✕ **0.3%(制限税率)**

- **200㎡以下**の部分は**評価額を3分の1**に
- **200㎡超**の部分は**評価額を3分の2**に

### 譲渡時の税金

土地建物の譲渡は**譲渡所得による分離課税**となる。
税率などの詳細は[タックスプランニング](./fp2-04-tax.md#譲渡所得)を参照。

譲渡所得 = 譲渡収入 - (取得費 + 譲渡費用)

- 取得費: **購入代金、購入時の仲介手数料**、不動産取得税、登録免許税
  - 不明な場合は**譲渡収入✕5%** で計算
  - 建物では減価償却後の金額を使う
  - 相続や贈与なら、非相続人や贈与者の取得費を**引き継ぐ**
  - **相続税の取得費加算の特例**がある
    - 相続した財産を3年以内に売った場合、相続税を取得費に加算できる
- 譲渡費用: **譲渡時の仲介手数料**、立退料、印紙税、解体費

#### 居住用財産の特例 (譲渡益)

いずれも必ず確定申告が必要。
**譲渡相手が身内でない**ことや、居住しなくなってから
**3年経過後の12月31日**までに譲渡していることなどが条件。

譲渡益がある場合、住宅ローン控除の併用は不可。

- **3000万円**の特別控除
  - **所有期間は問わない**
  - 軽減税率と併用可能
- **軽減税率**の特例 (軽減税率)
  - 所有期間が**10年超**であること
  - 特別控除後の金額に対し、**6000万円以下**の部分について15%5%ではなく**10%4%** に軽減
- **買換え**の特例 (課税繰り延べ)
  - 所有期間が**10年超**であること
  - 譲渡対価が**1億円以下**であること
  - 先の2つとは**併用不可**
  - 新しい家が古い家より高い場合は、**譲渡はなかったもの**として課税繰延べ
  - 新しい家が古い家より安い場合は、その**差額割合分だけ課税**し残りは繰延べ

#### 居住用財産の特例 (譲渡損)

所有期間が **5年超** の居住用財産を譲渡して損失が生じた場合、
**損益通算** や **繰越控除**（3年間、所得3000万以下が条件）が認められる。

譲渡損があっても、住宅ローン控除は併用可能。

- 買換える場合の特例
  - **10年以上の住宅ローン** を新たに組んでいること
  - 譲渡損失の全額を損益通算・繰越控除できる
- 買換えない場合の特例
  - 譲渡資産に **住宅ローンの残高** があること
  - 以下のいずれか **少ない方の金額** を損益通算・繰越控除できる
    - 住宅ローン残高 − 譲渡収入金額
    - 譲渡損失の金額
  - ローン残高が少ないと控除額も少なくなる。逆に、ローンを目一杯長くしておけば、買い替え不要で譲渡損失の全額を相殺できるといえる。

#### 空き家の特例

- 要件
  - 被相続人の居住用だった物件を、相続後に譲渡すること
  - 相続から**3年後の12月31日**までに譲渡すること
  - 評価額が**1億円以下**であること
  - 区分所有建物でないこと
  - 被相続人以外の居住者がいないこと
  - 1981年5月31日以前に建築されたものであること
  - 新耐震基準を満たすリフォームをするか、更地にして譲渡すること
- 控除額
  - **3000万円**
- 備考
  - 相続税の取得費加算の特例とは**併用不可**

## 不動産投資の有効活用と投資分析

### 土地の有効活用

- (自分でやる) 自己建設方式
  - **土地を所有したまま**、すべて**自分で行う**
- (業者におまかせ) 事業受託方式
  - **土地を所有したまま**、活用だけをデベロッパーに任せる
  - 資金調達は土地所有者が行う
  - 土地名義・建物名義とも**自分**となる
  - 所有者の**業務負担が少ない**一方、**収益性は低くなる**
- (信託でやる) 土地信託方式
  - 一定の期間、名義を預ける
  - 知識もお金も不要な一方、収入の保証はない
- (土地で出資) 等価交換方式
  - **土地を出資**したうえで、完成した土地と建物を出資比率で所有する
  - つまり**土地と建物を等価で交換**したのと同じといえる
  - **資金負担がない**
- (一定期間放置) 定期借地権方式
  - **定期借地権を設定**して地代を受け取る
  - **建設資金の負担が少なく**、長期の安定収入が見込める一方、転用は難しくなる
- (テナントからカツアゲ) 建設協力金方式
  - **建設協力金(保証金)** をテナントから**借りて**、**土地所有者名義で店舗等を建てる**
  - 賃料から協力金分を控除する
  - 権利関係がシンプル、貸家建付地なので相続に有利、一方で撤退リスクあり

### 賃貸住宅管理業法

サブリース業に関するルールが定められている。
住宅所有者とサブリース業者の間の契約をマスターリース(特定賃貸借)契約という。
サブリース業者と入居者の間の契約をサブリース(転貸借)契約という。

- マスターリース契約時に**家賃の減額があることを告げる義務**
- マスターリース契約締結前に**条件を記載した書面を交付して説明する義務**

### 不動産投資の収益性

投資額に対する収入を投資利回りという。これを測る指標がいくつかある。

- 表面利回り・単純利回り
  - 費用を無視した利回り
  - 表面利回り = 年間収入 / 投資総額
- 純利回り・NOI利回り・実質利回り
  - 費用を考慮した利回り
  - 純利回り = (年間収入 - 年間費用) / 投資総額
- キャッシュオンキャッシュ
  - 突っ込んだ額に対していくらキャッシュが返ってきているか
  - **借入金利よりも収益率が高ければCoCは高くなる**
  - CoC = 現金手取り / 自己資本
- 収益還元法
  - 将来生み出すであろうあらゆる収益を洗い出し、それらを**現在価値**に割り戻したうえで足し合わせることで、価格を求める方法
  - **直接還元法**
    - 単年度収益をシンプルに還元利回りで割り戻す
    - 価格 = (年間収入 - 年間経費) / 還元利回り
  - **DCF(Discounted Cash Flow)法**
    - 購入から売却までの純収益(**収益 - コスト + 復帰価格**)を現在価値に割り戻して比較する
    - 以下の2つの方法があるが、やっていることはどちらもほぼ同じ
      - **NPV(正味現在価値)法**
        - **金額**がプラスなら有利と判断する
        - **正味現在価値**とは、将来のすべての純収益から初期投資額を控除した金額のこと
        - 金利やインフレ率は考慮していない
      - **IRR(内部収益率)法**
        - 将来のすべての純収益の情報をもとに、正味現在価値がゼロになる割引率を算出し、その**率**が目標率を上回れば有利と判断する
        - 金利やインフレ率との比較が容易
- DSCR(借入金償還余裕率)
  - DSCR(倍率) = 年間キャッシュフロー / 年間返済額
  - 1倍よりも大きければ返済余力があるとみなされる

「総収入」と「純収益」の違いに注意する。

### 不動産の小口化と証券化

不動産の小口化とは、複数の投資家で一つの不動産に投資すること。

不動産の証券化とは、不動産収益を原資に社債や株式を発行すること。
投資家から集めた資金でビルやマンションなどに投資する**J-REIT**が有名。
