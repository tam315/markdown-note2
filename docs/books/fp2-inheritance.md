# FP2級 / 相続・事業継承

## 相続の基礎知識

**相続**とは、被相続人の財産を相続人に引き継ぐこと。

### 相続人

- **血族**とは、血の繋がった、互いに扶養義務のある一族のこと。
- **尊属(そんぞく)**とは、本人から見て、父母・祖父母など、前の世代に属する血族のこと。
- **卑属(ひぞく)**とは、本人から見て子供、孫、ひ孫など、後の世代に属する血族のこと。

**民法上**の相続人は、**被相続人の配偶者と一定の血族**である。
これらの人を**相続人**といい、以下の順序で決まる。
前提として、**配偶者は必ず相続人**になる。
法律上の配偶者でなければならず、内縁関係では相続人になれない。

- 配偶者
- 配偶者 + 子 (第1順位)
- 配偶者 + 直系尊属 (第2順位)
- 配偶者 + 兄弟姉妹 (第3順位)

相続人になれる子には、**養子や非嫡出子**も含まれる。

- 胎児
  - 被相続人が死んだ時点で生まれていない子
- 非嫡出子
  - 法律上は夫婦じゃない男女間の子
  - 被相続人が男なら認知が必要
- 普通養子
  - 実父母との関係を残した養子縁組のこと
  - **実父母・養父母の相続人になれる**
- 特別養子
  - 実父母との関係を終了して行なう養子縁組のこと
  - **養父母の相続人になれる**

以下の場合は相続人になれない。

- 死亡: 相続時に既になくなっている
- **欠格**: 殺害・脅迫により遺言状を書かせるなど
- **廃除**: 被相続人が家庭裁判所に申し立てた人(虐待をする者など)

死亡・欠格・廃除により相続人が相続権を失っている場合、
その人の子が変わりに相続することをができる。これを**代襲相続**という。
ただし、おい・めいが相続人の場合は、その子には代襲相続できない制約がある。
直系卑属の場合は制約はない。

寄与分とは、被相続人の世話をしてきた相続人(**寄与者**)への加算分。

被相続人の世話をしてきたけど法律上の相続人でない人を**特別寄与者**といい、
相続人に**特別寄与料を請求できる権利**がある。

被相続人から、生前贈与などで利益受けていた人を**特別受益者**といい、
受けた利益を**特別受益**という。

被相続人の所有する建物に住んでいた配偶者は、**配偶者居住権**を設定できる。
**終身**使用・収益することができる。配偶者がなくなると権利は消滅する。
通常の所有権より価値を低く見積もれるため、より多くの現金資産などを相続できるメリットもある。

所有権を相続しなくても、少なくとも6ヶ月は居住できる**配偶者短期居住権**もある。
無償かつ無条件で得られる権利であり、この部分に**相続税はかからない**。

### 相続分

相続人への割当のことを**相続分**と呼ぶ。

民法で定められた相続分を**法定相続分**といい、以下のように定められている。

- 配偶者がいる場合
  - 第1順位：配偶者は残りすべて、子は1/2
  - 第2順位：配偶者は残りすべて、直系尊属は1/3
  - 第3順位：配偶者は残りすべて、兄弟姉妹は1/4
- 配偶者がいない場合
  - 該当順位の相続人で均等に分割
- 半血兄弟姉妹の特則
  - 半血兄弟姉妹の相続分は、全血兄弟姉妹の 1/2

被相続人が遺言で指定した割合を**指定相続分**という。
法定相続分よりも指定相続分が優先される。

### 遺産分割

遺産を相続人で分割することを**遺産分割**という。

- 指定分割
  - 遺言により分割する方法
  - **協議分割よりも優先**される(モメたら被相続人の言うことがゼッタイ)
- 協議分割
  - 協議により相続人全員の合意のもと分割する方法
  - 全員で共謀すれば遺言に背く分割ができるということ
  - **法定相続分よりも優先**される
- 調停分割
  - モメたときに**家庭裁判所の調停**によって分割する方法
  - あくまで**相続人の合意が必要**
- 審判分割
  - 調停ですら決まらない場合に**家庭裁判所の審判**で分割する方法

分割にはいくつかの方法がある。

- 現物分割
  - 現物のまま、形状を変更することなく分割する方法
- 換価(かんか)分割
  - 不動産等を現金化して分割する方法
  - ケースによっては譲渡所得税がかかる
- 代償分割
  - ある相続人が特定の不動産等を相続したうえ、他の相続人に相応の現金などの**代償財産**を払ってバランスを取る方法
  - ケースによっては譲渡所得税がかかる

遺言がない、もしくは遺言と異なる分割をするときは、
以下の内容を記載した**遺産分割協議書**を作る。書式に決まりはない。

- 被相続人の情報
- 遺産分割内容
- 遺産分割内容に相続人が合意している旨
- **相続人全員の記名と押印**
- (代償分割を行う場合) 支払期日や支払方法

相続には負債も含まれるので、相続人はいくつかの選択肢を選べる。
単純承認がデフォで、それ以外を選ぶには家裁に対し**3ヶ月以内の申述が必要**。

- 単純承認
  - 財産をすべて相続する
- **限定承認**
  - プラスの資産の範囲内で、マイナスの資産を相続する
  - **相続人全員**で家裁に申述する
- **相続放棄**
  - すべて相続しない
  - **放棄する相続人が単独**で家裁に申述できる

### 遺言

死後の財産についての意思表示のことを**遺言**といい、
それを書面にしたものが**遺言書**である。
遺言によって相続人に無償で相続することを**遺贈(いぞう)**という。

- **満15歳以上**であれば誰でも作れる
- いつでも変更できる
- 複数の遺言書があって内容に差異がある場合、その部分については**新しい遺言書の記載が有効**になる
- 5年以内の期限を定めて遺産の全部または一部について、分割を禁止できる

**検認**とは、遺言書の存在と内容を明確にすることで、偽造・隠匿・消失等を防ぐための仕組み。
相続発生時に速やかに家裁にて手続きを行なう必要がある。
ただし、公正証書遺言や、**法務局保管の自筆証書遺言**の場合は、前述の恐れがないため、不要である。
検認は**遺言書の有効性を認めるものではない**ので注意。

遺言の方法は3つある。

- 自筆証書遺言
  - 全文を手書き＆日付記名押印
  - 財産目録だけはパソコンでもOKだが、**ページごとに署名と押印**が必要
  - 証人は**不要**
  - 保管は遺言者が行なう or 法務局の保管制度を利用する
- 秘密証書遺言
  - **パソコン作成や代筆も可**＆署名押印
  - 証人は**2人以上必要**
  - 保管は遺言者が行なう
- 公正証書遺言
  - 公証人が口述を筆記する
  - **保管は公証役場**が行なう
  - 証人は**2人以上必要**

**証人になれない人**は以下の通り。

- 未成年者
- 相続や遺贈を受ける可能性のある人や、これらの配偶者および直系血族
- 公証人の配偶者ならびに4親等内の親族等
