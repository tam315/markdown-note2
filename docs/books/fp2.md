## タックス・プランニング

### 税金の分類と所得税の基本

分類の軸は3つある。

- 誰が課税するか (国税 / 地方税)
- 税負担者と納税者が同じか (直接税 / 間接税)
- 何に税負担を求めるか (所得課税 / 消費課税 / 資産課税)

税額を自分で計算するのが**申告納税方式**である。
税額を国などが計算するのが**賦課課税方式**である。

収入から経費を引いたものを**所得**という。

所得税の**納税義務者**の住み方によって、課税される所得の種類がが変わる。

- 非居住者
  - 国内に住所がない個人（ただし継続して現在まで1年以上住んでいる居所がある場合は除く）
  - 国内源泉所得のみに課税される
- 居住者 (非永住者)
  - 国籍がなく、過去10年において、1年以上5年以下の間、住所または居所を持っている個人
  - 国内源泉所得と、国内で支払われたか国内に送金された国**外**源泉所得
- 居住者 (永住者)
  - 前述の2つ以外の個人
  - 国内・国外のすべての所得に課税される

所得税の**納税地**は原則として住所地になり、住所がない場合は居所地になる。
特例として、国内に住所と居所が両方ある場合は、居所を納税地とするとができる。
事業所がある場合はそこを納税地にできるが、住所または居所が国内にあることが前提条件である。

個人の**所得の種類**には10の分類がある。

- 利子所得
- 配当所得
- 不動産所得
- 事業所得
- 給与所得
- 退職所得
- 山林所得
- 譲渡所得
- 一時所得 (雑と異なり50万の控除あり)
- 雑所得

**非課税**となる所得もある。

- 社会保険の給付金
  - 年金、健康保険、失業手当など
- 損害保険・生命保険の給付金の一部
  - 身体の障害に基因するものなど
- 生活用動産の譲渡益
  - 家具、衣類など
  - 30万を超える貴金属などは除く
- 通勤手当
  - 15万円まで
- 宝くじの当選金

所得税の計算は原則として**総合課税**で行うが、一部の所得は独自の税率で計算される**分離課税**で行う。

分離課税には、申告の必要がある**申告分離課税**と、申告の必要がない（そもそもできない）**源泉分離課税**がある。

- 総合課税
  - 不動産
  - 事業
  - 給与
  - 一時(その他)
  - 雑(その他)
  - 譲渡(その他)
  - 利子(一般公社債)
  - 配当(非上場)
- 申告分離課税
  - 退職
  - 山林
  - 一時(懸賞金つき定期預金などごく一部)
  - 雑所得(FXなど)
  - 譲渡(土地・建物・株)
  - 利子(特定公社債)
  - 配当(上場)
- 源泉分離課税
  - 貯金の利子

総合課税の対象となる所得を合計したものを**総所得金額**という。

所得税は**超過累進税率方式**により税率が決まる。最低5%から最高45%まである。
