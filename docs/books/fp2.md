# FP2級

## タックス・プランニング

### 税金の分類と所得税の基本

分類の軸は3つある。

- 誰が課税するか (国税 / 地方税)
- 税負担者と納税者が同じか (直接税 / 間接税)
- 何に税負担を求めるか (所得課税 / 消費課税 / 資産課税)

税額を自分で計算するのが**申告納税方式**である。
税額を国などが計算するのが**賦課課税方式**である。

収入から経費を引いたものを**所得**という。

所得税の**納税義務者**の住み方によって、課税される所得の種類がが変わる。
前提として、**居住者**とは、国内に住所(≒住民票)があるか、現在まで継続して1年以上住んでいる場所があること。

- 非居住者
  - 国内源泉所得のみに課税される
- 居住者 (非永住者)
  - 居住者のうち、国籍がなく、かつ過去10年で日本居住が5年以下の個人
  - 国内源泉所得と、国内で支払われたか国内に送金された国**外**源泉所得
- 居住者 (永住者)
  - 居住者のうち、国籍があるか、過去10年で5年超日本に居住している個人
  - 国内・国外のすべての所得に課税される

所得税の**納税地**は原則として住所地になり、住所がない場合は居所地になる。
特例として、国内に住所と居所が両方ある場合は、居所を納税地とするとができる。
事業所がある場合はそこを納税地にできるが、住所または居所が国内にあることが前提条件である。

個人の**所得の種類**には10の分類がある。

- 利子所得
- 配当所得
- 不動産所得
- 事業所得
- 給与所得
- 退職所得
- 山林所得
- 譲渡所得
- 一時所得 (雑と異なり50万の控除あり)
- 雑所得

**非課税**となる所得もある。

- 社会保険の給付金
  - 年金、健康保険、失業手当など
- 損害保険・生命保険の給付金の一部
  - 身体の障害に基因するものなど
- 生活用動産の譲渡益
  - 家具、衣類など
  - 30万を超える貴金属などは除く
- 通勤手当
  - 15万円まで
- 宝くじの当選金

所得税の課税方式はいくつかある。原則として**総合課税**で行うが、一部の所得は独自の税率で計算される**分離課税**で行う。
さらに分離課税には、申告の必要がある**申告分離課税**と、課税が完全に完了して申告がそもそもできない**源泉分離課税**がある。

ややこしいが、**申告不要制度**というものもある。

- 総合課税
  - 不動産
  - 事業
  - 給与
  - 一時(その他)
  - 雑(その他)
  - 譲渡(その他)
  - 利子(一般公社債)
  - 配当(非上場)
- 申告分離課税
  - 退職
  - 山林
  - 一時(懸賞金つき定期預金などごく一部)
  - 雑所得(FXなど)
  - 譲渡(土地・建物・株)
  - 利子(特定公社債)
  - 配当(上場)
- 源泉分離課税
  - 利子(預貯金)

総合課税の対象となる所得を合計したものを**総所得金額**という。

所得税は**超過累進税率方式**により税率が決まる。最低5%から最高45%まである。

### 各所得金額の計算

#### 利子所得

経費は常にゼロで計算される。
課税方式は3種類もある。

- 預貯金の利子 -> 源泉分離課税
- 特定公社債の利子、公社債投資信託(株なし投信)の収益分配金 -> 申告分離課税
- その他の利子(一般公社債の利子など) -> 総合課税

#### 配当所得

株式の配当金や、株式投資信託(株あり投信)、ETF、J-REITの収益分配金が該当する。

`配当所得 = 収入 - 株式等取得のための負債利子`

課税方式は、3つある。

- 総合課税
  - 唯一、配当控除が受けられる
- 申告分離課税
  - 唯一、株式等譲渡益と損益通算ができる
- 申告不要制度
  - 唯一、確定申告が不要

原則は総合課税である。
ただし、3%以上の大口株主でなく、かつ**上場**株式等における配当所得については、後者2つのいずれかを選択できる。
なお、上場株式等の配当金は必ず源泉徴収された状態で支払われる。

#### 不動産所得

不動産の**貸付**による所得。
マンションや駐車場の賃料、礼金、更新料など。

ただし、ものによっては事業所得や雑所得になるので注意。
例えば、食事付き下宿の家賃、時間貸し駐車場の料金、従業員寮の家賃など。
なお、事業的規模であるかどうかは、所得の分類には無関係である。

以下の計算式にもとづき、**総合課税**となる。

`不動産所得 = 総収入金額 - 必要経費 (- 青色申告特別控除)`

- 総収入金額
  - 家賃、地代、礼金、更新料など
  - 支払期限を過ぎている未払家賃も含む
  - 入居者に返還しないことが決定済みの敷金等も含む
- 必要経費
  - 固定資産税をはじめとする税金、減価償却費、修繕費、借入金の利子など

#### 事業所得

事業から生じた所得のこと。総合課税となる。

`事業所得 = 総収入金額 - 必要経費 (- 青色申告特別控除)`

- 総収入金額
  - 年内に確定した未収入金も含む
- 必要経費
  - 売上原価
  - 減価償却費
  - 給与、通信費、水道光熱費、交際費など

売上原価は、その期に売り上げた商品の原価のこと。以下の式で計算される。

`売上原価 = 期首棚卸高 + 年間仕入高 - 期末棚卸高`

減価償却には定額法と定率法があり、資産の種類ごとにどちらかを選べる。
個人では、選択しなければ**定額法**が適用される。
なお、1998/4/1以降に取得した**建物**、2016/4/1以降に取得した**建物附属設備、構築物**は、定額法のみ選択可能。

減価償却には、資産の金額によっていくつかの特例がある。

- 少額減価償却資産
  - 10万円未満 or 試用期間が1年未満
  - 減価償却せず、全額をその年の経費にできる
- 一括償却資産
  - 10万円以上20万円未満
  - 3年間で均等償却できる
- 中小企業者等の少額減価償却資産の特例
  - 10万円以上30万円未満
  - 年間300万円まで
  - 減価償却する際、全額をその年の経費にできる

#### 給与所得

会社などから支払われる給与や賞与による所得。総合課税となる。

`給与所得 = 給与収入金額 - 給与所得控除額(65-195万円)`

源泉徴収 + 年末調整で納税が完了する場合、確定申告は不要である。
ただし、以下の場合は確定申告が必要。

- 給与所得が2,000万円を超える場合
- 給与所得以外の所得が20万円を超える場合
- 2か所以上から給与をもらっている場合

以下の場合場合には、**所得金額調整控除**というものがある。
計算式は`給与収入-850万 * 10%`で、上限は15万円である。

- 給与と年金を同時にもらっている場合
- 給与が850万円を超える者のうち、23歳未満の扶養親族がいたり、家族や本人が特別障害者であるなどの場合
