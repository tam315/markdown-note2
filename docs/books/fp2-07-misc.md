# FP2級 / その他

## 土地建物に関する特例等のまとめ

- 不動産取得税
  - 土地 宅地なら評価額1/2
  - 土地 宅地なら床面積の2倍面積相当分の税額を控除
  - 建物 評価額から1200万控除
- 固定資産税
  - 土地 200㎡以下の住宅用地 評価額 1/6
  - 土地 200㎡超の住宅用地 評価額 1/3
  - 建物 新築120㎡まで 税額 1/2 (戸建て3年、マンション5年、長期優良+2年)
- 相続税
  - 土地 居住用 330㎡ 80%
  - 土地 事業用 400㎡ 80%
  - 土地 賃貸用 200㎡ 50%
- 贈与税
  - 土地建物 配偶者控除 20年以上 2000万控除 一生に一回
  - 土地建物 直系尊属からの住宅資金 1000万控除
- 譲渡所得
  - 益
    - (1) 建物 居住物件 譲渡益から3000万を控除
    - (2) 建物 10年以上居住物件 6000万以下の部分に軽減税率
    - (3) 建物 10年以上居住物件 課税繰り延べ 1,2と併用不可
    - (4) 建物 空き家 譲渡益から3000万を控除 5と併用不可
    - (5) 建物 相続物件 相続税分を譲渡益から控除 4と併用不可
  - 損
    - 建物 5年以上居住物件 新たに10年ローンで買換え 損益通算と繰越控除
    - 建物 5年以上居住物件 ローン残高あり 損益通算と繰越控除
